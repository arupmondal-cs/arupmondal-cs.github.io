Research.

Construction and Implementation Study of Verifiable Delay Function

Advisor: Dr. Debayan Gupta, Ashoka University

A delay function is an important cryptographic technique used for adding delay in decentralized applications. Starting with the implementation of cryptocurrency, the last decades have witnessed numerous constructions of delay functions involving various mathematical techniques and a number of exciting applications. In this work, we present an efficient Verifiable Delay Function construction and its implementation study.

Secure Computation Framework for Privacy-Preserving Machine Learning

Advisor: Dr. Debayan Gupta, Ashoka University

Machine learning is widely used in practice to produce predictive models for applications such as image processing, speech and text recognition. These models are more accurate when trained on large amounts of data collected from different sources. However, the massive data collection raises privacy concerns. In this work, we present a simple, robust, and deployable framework for training a neural network using private data from multiple sources, using secret-shared secure function evaluation.

Theoretical and Practical Aspects of Secure Multiparty Computation

Advisor: Dr. Debayan Gupta, Ashoka University

Secure Multiparty Computation (MPC) enables a set of parties to interact and compute a joint function of their private inputs while revealing nothing but the output. The potential applications for MPC are huge: privacy-preserving auctions, private DNA comparisons, private machine learning, threshold cryptography, and more. My interest is, to present some improved theoretical results and build an efficient MPC framework.

Feasibility and Degree of Achievable Traceability in End-to-End Encryption

Advisor: Dr. Debayan Gupta, Ashoka University

The purpose of this project is to objectively analyse current end-to-encryption schemes (given current circumstances, we focus on the Signal protocol) used in chat messaging applications such as WhatsApp, to determine whether and to what degree traceability features may be introduced into the protocols without compromising security. We believe that all government agencies, and chat messaging applications are working towards the benefit of the public, and through this document we are providing a statement of facts and limitations of current technologies to incorporate suggestions. We do not aim to intentionally condemn the requests, practices, or suggestions of any individual, organization, or government agency.

Estimating Miss Ratio from the Characteristic of an Algorithm

Advisor: Prof. S. K. Basu, Banaras Hindu University (July 2017 - April 2018)

M.Sc. Thesis: To estimate miss ratio from the characteristics of an algorithm using a simple model of the cache. The model is validated by trace-driven simulations of several program traces and formulate the mathematical form and improve the measurement of the program behavior and complexity of the programs.

Advisor: Prof. S. K. Basu, Banaras Hindu University (January 2018 - June 2018)

I worked on the memory hierarchy of the systems, in particular, we looked at the memory reference pattern of the programs and characterized the behavior, simulation complexity, and the memory reference pattern of the programs.
